**The actual schema of the Lighthouse object is liable to change depending on the page and Lighthouse version.**

## Schema

### `lighthouseVersion`

The version of Lighthouse used to generate the report.

### `requestedUrl`

The URL that was requested for testing.

### `audits`

An object containing the results of all audits performed. Each audit is identified by its ID (e.g., `largest-contentful-paint`) and contains details such as `title`, `numericValue`, and `score`.

### `categories`

An object containing the overall scores for each Lighthouse category, such as Performance and Accessibility. Each category has a `title` and a `score` ranging from 0 to 1.

### `categoryGroups`

Information about how audits are grouped within the report.
